# 学生宿舍管理系统 python527

## 项目概述

学生宿舍管理系统是基于 Python 语言开发的一套管理软件，旨在提高宿舍管理的效率与准确性。

## 技术栈

- **后端**: Python 3.7.7, Django
- **前端**: Vue
- **数据库**: Mysql 8
- **开发工具**: PyCharm
- **操作系统**: Windows 10, MacOs

## 系统架构

- **架构模式**: MVC

## 环境配置

- **数据库**: Mysql 8
- **数据库可视化工具**: Navicat

## 功能模块

系统主要包括以下功能模块：

- 学生信息管理
- 宿舍楼信息管理
- 宿舍分配管理
- 维修与投诉处理

## 系统角色

- 管理员：负责系统的维护与宿舍分配管理
- 学生：查看个人信息、宿舍信息，提交维修与投诉

## 运行环境

- Python 3.7.7
- Mysql 8
